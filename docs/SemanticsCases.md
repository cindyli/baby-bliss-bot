# Semantics Cases

## Gerunds

A **gerund** functions as a noun, while having the verb ending -ING. Gerunds represent an action rather than a thing. In Blissymbolics, there are four ways in which gerunds could be applied.
1) without the continuous indicator [28043]
2) with the continuous indicator [28043]
* Example 1. Sing [16990] + [28043]
3) context-specific strategies
* Example 1. activity (classifier) [12335] + run (specifier) [26234]
* Example 2. LEGS [15189] + arrow forward [14390] + FAST (quickly) [16446]
* Example 3. running (sport) [21839]
4) with the present tense [8994]
* Example 1. Sing [16990] + [8994]
